Reset NGC Folder Tool

Repair Windows Hello PIN and Face recognition issues after Windows updates with a single script.

Overview

This script resets the Windows Hello NGC folder, which can become corrupted after updates. Running it allows you to reconfigure your PIN and Face recognition easily.

Warning: Running this script will remove your Windows Hello PIN and biometric data. You will need to set up your sign-in methods again.
